Secure File Encryption Application (Group Project)

Designed and implemented a desktop application enabling file encryption/decryption using AES-256 with a user-friendly graphical interface.

Developed account management features, including password recovery and client-server authentication via RSA digital signatures to ensure secure user access.

Built a challenge-response authentication system to prevent man-in-the-middle attacks and safeguard remote communications.

Implemented storage separation by user email to enhance file confidentiality and account-level data isolation.

Delivered a lightweight, intuitive alternative to existing encryption tools, balancing strong security protocols with usability for non-technical users.

Player Dodge Game

This is a game where you have to avoid all of the obstacles that are thrown at you like fireballs, cars, rockets, and cannons for 30 rounds. There are 10 different types of obstacles including 5 different fireball colors, 2 different car and rocket sizes, and cannons with lots of animation and colors. Built with Python using the CMU CS Academy website module.

Instructions: Avoid the obstacles by pressing the arrow keys to move the player. You can also collect the snowflakes on the screen to gain points for each round. The player will have to survive for 30 rounds or gain 30 points to win the game.
